Implementation of Dijkstra's Algorithm

This program contains :

the main file : main.cpp

the include directory : include

  • utilities vector functions and some type definitions : util.h
  • Header file of close set : Close.h
  • Header file of edge : Edge.h
  • Header file of graph : Graph.h
  • Header file of open set : Open.h
  • Header file of shortest patth : ShortestPath.h

the source directory : scr

  • Implementation of close set : Close.cpp
  • Implementation of edge : Edge.cpp
  • Implementation of graph : Graph.cpp
  • Implementation of open set : Open.cpp
  • Implementation of shortest patth : ShortestPath.cpp

The compiler must enable C++11 feature in order to compile the program
